| Проект              | Описание           | Библиотеки                     |

| [Исследование надежности заемщиков](Исследование надёжности заёмщиков) | Определить влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок | *pandas*|

| [Прогноз продаж в интернет-магазине](Прогноз продаж в интернет-магазине) | Выявление связей и закономерностей, определяющих успешность игры. | *pandas*, *numpy*, *matplotlib*, *seaborn*, *scipy* |

| [Выбор локации для нефтяной скважины](Выбор локации для нефтяной скважины) | Построение модели для определения региона с наибольшей прибылью от добычи. | *pandas*, *numpy*, *matplotlib*, *seaborn*, *sklearn* |
| [Восстановление золота из руды](Восстановление золота из руды) | Построение модели для предсказания коэффициента восстановления золота из золотосодержащей руды. | *pandas*, *numpy*, *matplotlib*, *seaborn*, *sklearn* |
| [Определение стоимости автомобилей](Определение стоимости автомобилей) | Определение стоимость автомобилей по заданным характеристикам. | *pandas*, *numpy*, *sklearn*, *catboost*, *lightgbm* |
| [Прогнозирование заказов такси](Прогнозирование заказов такси) | Прогноз заказов такси на следующий час. | *pandas*, *numpy*, *statsmodels*, *matplotlib*, *sklearn*, *catboost*, *lightgbm* |
| [Прогнозирование оттока клиентов телекоммуникационной компании](Прогнозирование оттока клиентов телекоммуникационной компании) | Построение модели по прогнозированию оттока клиентов. | *pandas*, *numpy*, *matplotlib*, *seaborn*, *sklearn*, *catboost*, *phik* |
